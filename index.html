Guida all'uso
🌿 Panoramica
+
Biodanza Manager è un'app per la gestione di una palestra di Biodanza. Permette di registrare i partecipanti, documentare ogni sessione, monitorare le presenze e raccogliere feedback. Tutti i dati sono salvati localmente sul dispositivo (localStorage). L'app è accessibile tramite browser dopo il deploy su GitHub Pages.
👤 Anagrafica Utenti
+
Contiene la lista di tutti i partecipanti. Per ogni utente puoi inserire: nome e cognome, telefono/WhatsApp, email, data di nascita (l'età viene calcolata automaticamente), data di iscrizione, certificato medico (Sì/No — mostrato come badge colorato nella lista), note libere.

+ Nuovo → apre il form di inserimento.
✎ → riapre il form precompilato per modificare i dati.
✕ → elimina l'utente (con conferma).
📅 Calendario
+
Vista mensile del calendario. Il giorno odierno è evidenziato in verde. I giorni con almeno una sessione registrata mostrano un pallino viola in basso. Sotto il calendario appare la lista delle sessioni del mese selezionato con tema e numero di presenti. Usa ‹ › per navigare tra i mesi.
🎵 Sessioni
+
Qui si registrano le sessioni. Per ogni sessione: data, tema (scelto dalla Biblioteca o scritto liberamente), presenti (spunta i partecipanti dalla lista anagrafica), note del conduttore, feedback raccolti a voce.

Sotto il form appare la lista di tutte le sessioni registrate in ordine cronologico inverso. Ogni sessione mostra tema, data, numero presenti e nomi. ✕ elimina la sessione.
📊 Dashboard
+
Panoramica statistica automatica aggiornata in tempo reale.

Tre metriche in evidenza: totale utenti, totale sessioni, totale presenze cumulative.
Grafico a barre: presenze mensili nell'anno in corso.
Frequenza per utente: barra di partecipazione di ogni utente (presenze / sessioni totali %).
Temi più usati: classifica dei 5 temi più frequenti nelle sessioni registrate.
📚 Biblioteca Tematica
+
Archivio dei temi Biodanza usati nelle sessioni. Ogni tema ha: nome, descrizione, esercizi suggeriti, musiche suggerite (solo testo descrittivo, non file audio).

I temi salvati qui appaiono nel menu a tendina quando si registra una nuova sessione. È un promemoria strutturato per il conduttore, non influenza le statistiche. + Tema aggiunge un nuovo tema. ✕ lo elimina.
💬 Feedback
+
Permette di raccogliere feedback dai partecipanti tramite un link condivisibile su WhatsApp.

Come funziona:
1. Scegli la sessione dal menu.
2. L'app genera un link URL univoco per quella sessione.
3. Premi Copia e incollalo su WhatsApp al gruppo.
4. Il partecipante apre il link dal telefono e trova: 8 risposte prefissate a scelta singola, voto da 1 a 5 stelle, nota libera opzionale.
5. Le risposte appaiono nella sezione Risposte ricevute.

Importante: il link funziona solo se l'app è pubblicata su GitHub Pages. Non funziona aprendo il file HTML direttamente dal computer.
🔒 Password e accesso
+
L'app è protetta da password. La sessione viene ricordata per 7 giorni: non è necessario reinserire la password ad ogni apertura. Dopo 7 giorni viene richiesta nuovamente. I dati sono salvati localmente sul dispositivo in uso — non vengono trasmessi a server esterni.
